# VANTA Automations Website

Premium SaaS landing website for VANTA Automations.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide icons

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Design Direction

The visual system uses a premium dark SaaS identity with subtle pixel accents:

- Gun Metal background: `#00272B`
- Dark Tiber panels: `#06392F`
- Chartreuse accent: `#E0FF4F`
- Premium glass cards, thin borders, soft glows
- Pixel particles, pixel corner cuts, terrain-inspired footer
